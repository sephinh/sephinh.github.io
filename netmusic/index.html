网易云音乐降级4.3.1 PLAY版

2022-11-05 2 min read # Do some useless

    虽然听歌已经有YouTube Music，Spotify，Tidal，但是网易云可能听的时间长了，推荐歌单很靠谱，虽说版权问题越来越难，但一直没放弃在用。Android端一直用的是PLAY版的4.3.1版本的网易云音乐，简介明了无更新提示，用的好好的，前两天突然登录掉了，然后网易邮箱再登录就提示让更新了，蛋疼之余只能找了个7.x的据说还行的版本先用上了，回头想adb降级一下就ok了，中间被每次打开的更新提示烦了好久，今天懒癌缓解赶快折腾。

    电脑上使用adb链接，结果手机那个同意usb调试的确定死活点不了，提示被遮挡，查找解决方案无非就是什么悬浮按钮啥的，使用无果，强行重启了一遍好了，
结果：*adb install -r -d app.apk*​，
命令提示失败，说什么sdk高版本降低版本有问题，好吧，那就保资料卸载，
查找网易云的包名：pm list packages：*com.netease.cloudmusic*，
然后：*adb shell pm uninstall -k com.netease.cloudmusic*，
保资料卸载后：*adb install 路径/apk 安装4.3.1* 版，
终于成功了，结果手机一点开，提示旧版应用，然后就闪退了，WTF,原来4.31一直在用只是不能登录，这不至于提示个旧版就不能用了啊，无奈清除软件缓存重新打开了，但是登录依然回到起点无解。。

    难不成还得低一点的能登录的版本降级才好使，上酷安寻找一番，正巧看见一老哥解决方案，他寻思网易总不能把所有版本都禁止登录了吧，果然让他找到个4.3.4版本的国内版，可以登录，但有更新提示，正好拿来降级用，下载安装登录，可以登录，保资料降级，终于打开了熟悉的4.3.1无更新提示的界面，终于折腾舒服了。。
